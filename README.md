# ENI Madagascar — Portfolio Web v2

> Portfolio officiel de l'École Nationale d'Informatique (ENI) de Fianarantsoa, Madagascar.
> Thème : **Vert** (principal) + **Rouge** (logo ENI).
> Données issues du site officiel **eni.mg**.

## Stack technique
- **React 18** + **Vite 5**
- **Tailwind CSS 3**
- **Framer Motion 11** (animations scroll, hover, transitions)
- **GSAP 3** (animations hero, ScrollTrigger)
- **React Router 6** (navigation SPA)
- **React CountUp** (compteurs animés)
- **React Type Animation** (effet machine à écrire)
- **React Icons** (icônes)

```bash

```

